Condicionales

Las estructuras condicionales te permiten ejecutar diferentes bloques de código basados en si una condición es verdadera o falsa.

  • IF:La instrucción if ejecuta un bloque de código si una condición especificada es verdadera.
    • Ejemplo If:
    
            let edad = 18;
            if (edad >= 18) {
            console.log("Eres mayor de edad.");
            }
  • ELSE: La instrucción else se ejecuta si la condición de la instrucción if es falsa.
    • Ejemplo else:
    
            let temperatura = 15;
                if (temperatura > 20) {
                console.log("Hace calor.");
                } else {
                console.log("Hace frío.");
                }
  • ELSE IF: Se utiliza para verificar múltiples condiciones en secuencia. Si la condición del if es falsa, se evalúa la condición del primer else if. Si es verdadera, se ejecuta su bloque de código. Puedes tener múltiples else if.
    • Ejemplo else if:
    
            let nota = 75;
            if (nota >= 90) {
            console.log("Sobresaliente");
            } else if (nota >= 70) {
            console.log("Notable");
            } else if (nota >= 60) {
            console.log("Aprobado");
            } else {
            console.log("Reprobado");
            }
  • SWITCH: La instrucción switch evalúa una expresión, comparando el valor del resultado con los valores de diferentes casos (case). Si hay una coincidencia, se ejecutan las instrucciones asociadas a ese case. La instrucción break se utiliza para salir del switch después de que se encuentra una coincidencia y se ejecuta el código correspondiente. Si no se encuentra ninguna coincidencia, se puede ejecutar un bloque de código default (opcional).
    • Ejemplo switch:
    
            let dia = "lunes";
            switch (dia) {
            case "lunes":
                console.log("Es el inicio de la semana.");
                break;
            case "viernes":
                console.log("¡Ya casi es fin de semana!");
                break;
            case "sabado":
            case "domingo":
                console.log("Es fin de semana.");
                break;
            default:
                console.log("Es un día entre semana.");
            }

Bucles

Los bucles te permiten ejecutar un bloque de código repetidamente hasta que se cumpla una determinada condición.

  • FOR:El bucle for es útil cuando sabes cuántas veces quieres que se ejecute un bloque de código. Consta de tres partes principales dentro de los paréntesis:

    • Inicialización:Se ejecuta una sola vez al principio del bucle (generalmente para inicializar una variable contador).

    Condicion:Se evalúa en cada iteración. El bucle continúa mientras la condición sea verdadera.

    Incremento/Decremento: Se ejecuta al final de cada iteración (generalmente para actualizar la variable contador).

    Ejemplo FOR:
    
            for (let i = 0; i < 5; i++) {
            console.log("El valor de i es: " + i);
            }
  • WHILE:El bucle while ejecuta un bloque de código mientras una condición especificada sea verdadera. La condición se evalúa antes de cada iteración
    • Ejemplo WHILE:
    
           let contador = 0;
            while (contador < 3) {
            console.log("Contador: " + contador);
            contador++;
            }
  • DO WHILE:El bucle do-while es similar al while, pero la condición se evalúa después de la primera ejecución del bloque de código. Esto garantiza que el bloque de código se ejecute al menos una vez
    • Ejemplo DO WHILE:
    
           let numero = 0;
            do {
            console.log("Número: " + numero);
            numero++;
            } while (numero < 2);
  • FOR EACH:El método forEach() se utiliza para iterar sobre los elementos de un array. Proporciona una función de callback que se ejecuta para cada elemento del array.
    • Ejemplo FOR EACH:
    
           let colores = ["rojo", "verde", "azul"];
            colores.forEach(function(color) {
            console.log("Color: " + color);
            });

            // También se puede usar con una función de flecha (arrow function)
            colores.forEach(color => console.log("Color (arrow): " + color));
