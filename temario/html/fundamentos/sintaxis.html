Declaraciones

En JavaScript, se utilizan palabras clave para declarar variables, lo que esencialmente reserva un espacio con un nombre en la memoria para almacenar datos.

  • var:Es la forma más antigua de declarar variables en JavaScript. Aunque todavía funciona, se recomienda usar let o const en el código moderno debido a su comportamiento más predecible en cuanto al alcance (scope) de las variables.
  • let: Se utiliza para declarar variables que pueden ser reasignadas con un nuevo valor. Es la forma preferida para la mayoría de las variables.
  • const: Se utiliza para declarar variables cuyo valor no debe ser reasignado después de la inicialización. Si intentas reasignar una variable const, obtendrás un error. No se puede declarar una constante con el mismo nombre que una función o variable en el mismo ámbito
Ejemplo de declaración de variables:

        // Declaración con var
        var nombre = "Carlos";

        // Declaración con let
        let edad = 30;

        // Declaración con const
        const PI = 3.1416;

Variables

Se utilizan como nombres simbólicos para valores en tu aplicación. Los identificadores deben comenzar con una letra, guión bajo (_) o signo de dólar ($). Luego pueden incluir dígitos.

Ejemplos válidos: Number_hits, temp99, $credit, _name.

Evaluar variables

Si se declara una variable con var o let sin asignar valor, su valor es undefined.

Acceder a una variable no declarada lanza una excepción ReferenceError.

Ámbito de variables

Las variables declaradas fuera de funciones son globales. Dentro de funciones son locales.

En páginas web, las variables globales son propiedades del objeto window. Por ejemplo, puedes acceder a window.phoneNumber desde un iframe con parent.phoneNumber.

Tipos de datos primitivos

ECMAScript define ocho tipos de datos:

  • String: Representa una secuencia de caracteres de texto. Se delimitan con comillas simples (') o dobles (").
  • Number: Representa valores numéricos, incluyendo enteros y números de punto flotante. JavaScript no tiene tipos separados para enteros y decimales.
  • BigInt: Enteros grandes (9007199254740992n)
  • Boolean:Representa un valor de verdad, que puede ser true (verdadero) o false (falso).
  • undefined:Representa un valor que no ha sido asignado. Una variable declarada sin una asignación inicial tiene el valor undefined. También es el valor que se devuelve cuando se intenta acceder a una propiedad inexistente de un objeto.
  • null:Representa la ausencia intencional de un valor. Es un valor literal que representa una referencia nula o "sin valor".
  • Symbol:Representa un valor único e inmutable. Se utiliza principalmente para crear propiedades de objeto privadas o para evitar colisiones de nombres de propiedades
  • Object: Colección de propiedades y funciones
Ejemplo de tipos primitivos:

    let nombre = "Ana";           // String
    let edad = 25;                // Number
    let esEstudiante = true;      // Boolean
    let identificador = Symbol(); // Symbol
    let datoIndefinido;           // undefined
    let valorNulo = null;         // null
    let numeroGrande = 9007199254740992n; // BigInt

Operadores

Los operadores son símbolos especiales que realizan operaciones en uno o más operandos (valores).

Operadores Aritméticos

Realizan operaciones matemáticas básicas.

  • +:Suma
  • -:Resta
  • *:Multiplicacion
  • /:Division
  • %:Módulo (resto de la división)
  • **:Exponenciación (ES2016)
  • ++:Incremento (prefijo o sufijo)
  • --:Decremento (prefijo o sufijo)
Ejemplo de operadores aritméticos:

    let a = 4, b = 2;
    console.log(a + b); // 6
    console.log(a - b); // 2
    console.log(a * b); // 8
    console.log(a / b); // 2
    console.log(a % b); // 0
    console.log(a ** 2); // 16
    a++;
    b--;
    console.log(a, b); // 5, 1

Operadores Logicos

Se utilizan para combinar o modificar expresiones booleanas.

  • &&:AND lógico (devuelve true si ambos operandos son true)
  • ||:OR lógico (devuelve true si al menos uno de los operandos es true)
  • !:NOT lógico (devuelve el valor booleano contrario del operando)
Ejemplo de operadores lógicos:
let esMayor = true;
    let esAlumno = false;

    console.log(esMayor && esAlumno); // false
    console.log(esMayor || esAlumno); // true
    console.log(!esAlumno);           // true

Operadores de comparación

Se utilizan para comparar dos operandos y devuelven un valor booleano (true o false).

  • ==: Igualdad (compara solo el valor, puede haber conversión de tipo)
  • ===:Igualdad estricta (compara tanto el valor como el tipo)
  • !=:Desigualdad (compara solo el valor)
  • !==:Desigualdad estricta (compara tanto el valor como el tipo)
  • >:Mayor que
  • <:Menor que
  • >=:Mayor o igual que
  • <=:Menor o igual que
Ejemplo de operadores de comparación:
let x = 5;
    let y = "5";
    console.log(x == y);  // true
    console.log(x === y); // false
    console.log(x != y);  // false
    console.log(x !== y); // true
    console.log(x > 3);   // true
    console.log(x <= 5);  // true
