FUNCIONES QUE DEVUELVEN FUNCIONES

Las funciones de orden superior son funciones que operan sobre otras funciones, ya sea tomándolas como argumentos o devolviéndolas.


            function crearMultiplicador(factor) {
            return function(numero) {
                return numero * factor;
            };}
            const duplicar = crearMultiplicador(2);
            const triplicar = crearMultiplicador(3);

            console.log(duplicar(5));   // Output: 10
            console.log(triplicar(5));  // Output: 15

FUNCIONES DE ORDEN SUPERIOR

Las funciones de orden superior son funciones que operan sobre otras funciones, ya sea tomándolas como argumentos o devolviéndolas.

  • map(callback): Ya lo vimos en el contexto de arrays, pero es importante recordar que map es una función de orden superior. Toma una función (callback) como argumento y la aplica a cada elemento de un array, devolviendo un nuevo array con los resultados.
    • 
            let numeros = [1, 2, 3];
            let dobles = numeros.map(function(numero) {
            return numero * 2;
            });
            console.log(dobles); // Output: [2, 4, 6]
            console.log(numeros); // Output: [1, 2, 3] (el array original no se modifica)

    Resultados:

  • filter(callback): Similar a map, filter también es una función de orden superior. Recibe una función (callback) que actúa como un predicado para cada elemento de un array. Devuelve un nuevo array con todos los elementos para los cuales el callback devuelve true.
    • 
            const edades = [15, 22, 18, 30, 12];
            const mayores = edades.filter(function(edad) { // La función anónima es el callback
            return edad >= 18;
            });
            console.log(mayores);

  • reduce(callback, valorInicial): reduce también es una función de orden superior. Toma una función (callback) que se aplica acumulativamente a los elementos de un array para reducirlos a un único valor. El callback recibe el acumulador y el elemento actual.
    • 
            const precios = [10, 20, 5];
            const total = precios.reduce(function(acumulador, precio) { // La función anónima es el callback
            return acumulador + precio;
            }, 0);
            console.log(total); // Output: 35
