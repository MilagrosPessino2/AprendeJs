📌 Variables en JavaScript

Una variable es un espacio en memoria que se utiliza para almacenar datos. JavaScript permite declararlas usando var, let y const.

  • var: Declaración clásica. Tiene alcance de función.
  • let: Se recomienda actualmente. Tiene alcance de bloque.
  • const: Para valores que no cambian. Tiene alcance de bloque.
// Ejemplo con let
let nombre = "Ana";
nombre = "Luis";

// Ejemplo con const
const PI = 3.1416;
// PI = 3.14; ❌ Error, no se puede reasignar

📌 Tipos de datos primitivos

  • String: Cadenas de texto. Ej: "Hola"
  • Number: Números. Ej: 5, 3.14
  • Boolean: true o false
  • Null: Valor intencionadamente nulo
  • Undefined: Variable declarada sin valor
  • Symbol: Valores únicos y no modificables

📌 Operadores

  • Aritméticos: +, -, *, /, %
  • Lógicos: &&, ||, !
  • Comparación: ==, ===, !=, !==, >, <, >=, <=

Estructuras de control

  • Condicionales (if, else, switch)
  • Bucles (for, while, do-while, forEach)

📌 Condicionales

let edad = 18;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

// switch
let dia = "lunes";
switch (dia) {
  case "lunes":
    console.log("Comienza la semana");
    break;
  case "viernes":
    console.log("Fin de semana cercano");
    break;
  default:
    console.log("Otro día");
}

📌 Bucles

// for
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// while
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

// forEach
[1, 2, 3].forEach(function(n) {
  console.log(n);
});

Funciones

  • Funciones tradicionales y anónimas
  • Arrow functions (=>)
  • Funciones callback y recursivas
  • Funciones autoejecutables

📌 Funciones tradicionales

function saludar(nombre) {
  return "Hola " + nombre;
}

📌 Funciones anónimas

const sumar = function(a, b) {
  return a + b;
};

📌 Arrow functions

const restar = (a, b) => a - b;

📌 Callback

function procesar(valor, callback) {
  return callback(valor);
}

procesar(5, function(x) {
  return x * 2;
});

📌 Recursiva

function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}

📌 Función autoejecutable

(function() {
  console.log("Se ejecuta automáticamente");
})();
