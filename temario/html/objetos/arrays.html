ARRAYS

Un array (también conocido como arreglo o vector en algunos contextos) es una estructura de datos que permite almacenar una colección ordenada de elementos, los cuales pueden ser de cualquier tipo (números, strings, booleanos, otros arrays, objetos, etc.). Cada elemento en un array tiene una posición numérica llamada índice, que comienza desde 0 para el primer elemento.

  • Metodos de arrays: Los lenguajes de programación modernos ofrecen una variedad de métodos integrados para manipular arrays de manera eficiente. Estos son los más comunes:

    • push: Agrega uno o más elementos al final del array y devuelve la nueva longitud del array.

    
            let frutas = ["manzana", "banana"];
            let nuevaLongitud = frutas.push("naranja");
            console.log(frutas); // Output: ["manzana", "banana", "naranja"]
            console.log(nuevaLongitud); // Output: 3

    pop: Elimina el último elemento del array y devuelve ese elemento eliminado. Si el array está vacío, devuelve undefined.

    
            let frutas = ["manzana", "banana", "naranja"];
            let ultimoElemento = frutas.pop();
            console.log(frutas); // Output: ["manzana", "banana"]
            console.log(ultimoElemento); // Output: "naranja"

    shift: Elimina el primer elemento del array y devuelve ese elemento eliminado. Si el array está vacío, devuelve undefined. Reindexa los elementos restantes.

    
            let frutas = ["manzana", "banana", "naranja"];
            let primerElemento = frutas.shift();
            console.log(frutas); // Output: ["banana", "naranja"]
            console.log(primerElemento); // Output: "manzana"

    unshift(elemento1, elemento2, ...): Agrega uno o más elementos al principio del array y devuelve la nueva longitud del array. Reindexa los elementos existentes.

    
            let frutas = ["banana", "naranja"];
            let nuevaLongitud = frutas.unshift("manzana", "kiwi");
            console.log(frutas); // Output: ["manzana", "kiwi", "banana", "naranja"]
            console.log(nuevaLongitud); // Output: 4

    map(callback(elemento, indice, array)): Crea un nuevo array con los resultados de llamar a una función proporcionada (callback) en cada elemento del array original. La función callback recibe tres argumentos: el elemento actual, su índice y el array original.

    
            let numeros = [1, 2, 3];
            let dobles = numeros.map(function(numero) {
            return numero * 2;
            });
            console.log(dobles); // Output: [2, 4, 6]
            console.log(numeros); // Output: [1, 2, 3] (el array original no se modifica)

    filter(callback(elemento, indice, array)): Crea un nuevo array con todos los elementos que pasan la prueba implementada por la función proporcionada (callback). Al igual que map, la función callback recibe el elemento, su índice y el array original, y debe devolver true para mantener el elemento en el nuevo array, o false para descartarlo.

    
            let numeros = [1, 2, 3, 4, 5];
            let pares = numeros.filter(function(numero) {
            return numero % 2 === 0;
            });
            console.log(pares); // Output: [2, 4]
            console.log(numeros); // Output: [1, 2, 3, 4, 5] (el array original no se modifica)

    reduce(callback(acumulador, elemento, indice, array), valorInicial): Aplica una función (callback) a un acumulador y a cada elemento del array (de izquierda a derecha) para reducirlo a un único valor. La función callback recibe el acumulador, el elemento actual, su índice y el array original. El valorInicial es un valor opcional para el acumulador en la primera llamada de la función callback. Si no se proporciona, se utiliza el primer elemento del array como acumulador y la iteración comienza desde el segundo elemento.

    
            let numeros = [1, 2, 3, 4];
            let suma = numeros.reduce(function(acumulador, numero) {
            return acumulador + numero;
            }, 0);
            console.log(suma); // Output: 10

            let multiplicacion = numeros.reduce((acumulador, numero) => acumulador * numero, 1);
            console.log(multiplicacion); // Output: 24
  • Acceso y Modificación de Arrays: Se puede acceder a los elementos de un array utilizando su índice dentro de corchetes []. Los índices comienzan desde 0. También se pueden modificar los elementos asignando un nuevo valor a una posición específica del array.

    • Acceso

    
            let colores = ["rojo", "verde", "azul"];
            console.log(colores[0]); // Output: "rojo"
            console.log(colores[2]); // Output: "azul"
            console.log(colores[3]); // Output: undefined (índice fuera de rango)

    Modificación

    
            let colores = ["rojo", "verde", "azul"];
            colores[1] = "amarillo";
            console.log(colores); // Output: ["rojo", "amarillo", "azul"]
  • Iteracion de Arrays: Existen varias formas de recorrer o iterar sobre los elementos de un array para realizar alguna operación en cada uno de ellos

    • For loop tradiciona: Es la forma más básica y explícita de iterar. Se controla el índice de forma manual.

    
            let numeros = [10, 20, 30];
            for (let i = 0; i < numeros.length; i++) {
            console.log(`El elemento en el índice ${i} es: ${numeros[i]}`);
            }

    Map: También se utiliza para iterar sobre cada elemento y realizar una acción (aunque su principal propósito es crear un nuevo array).

    
            let nombres = ["Ana", "Pedro", "Luisa"];
            nombres.map(function(nombre) {
            console.log(`Hola, ${nombre}!`);
            });

    ForEach: Ejecuta una función proporcionada (callback) una vez por cada elemento del array. Es similar a map en su forma de iterar, pero no crea un nuevo array; se utiliza principalmente para realizar efectos secundarios (como imprimir en la consola, modificar variables externas, etc.).

    
            let colores = ["rojo", "verde", "azul"];
            colores.forEach(function(color, indice) {
            console.log(`El color en la posición ${indice} es: ${color}`);
            });

    Filter: Aunque su propósito principal es filtrar elementos, también itera sobre cada elemento del array para aplicar la condición de filtrado.

    
            let edades = [15, 22, 18, 30, 12];
            let mayoresDeEdad = edades.filter(function(edad) {
            return edad >= 18;
            });
            console.log(mayoresDeEdad); // Output: [22, 18, 30]
