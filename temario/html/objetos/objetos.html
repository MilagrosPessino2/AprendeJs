Objetos (Tipo de Dato)

Los objetos en JavaScript son colecciones dinámicas de pares clave-valor. Son fundamentales para estructurar datos de manera organizada.

  • Crear objetos: Se pueden crear objetos utilizando la notación literal (llaves {}) o mediante constructores (como new Object() o clases).
    • 
          // Creación de un objeto literal
            const miObjeto = {
            nombre: "Producto A", // propiedad: valor
            precio: 25.99,
            disponible: true,
            mostrarDetalles: function() { // método: función dentro del objeto
                console.log(`Nombre: ${this.nombre}, Precio: $${this.precio}, Disponible: ${this.disponible}`);
            }
            };
            console.log(miObjeto);
            miObjeto.mostrarDetalles();
  • Acceso a propiedades (notación de punto y corchetes): Hay dos formas principales de acceder a las propiedades de un objeto:

    • Notación de punto (.):Se utiliza cuando conoces el nombre de la propiedad y este es un identificador válido (sin espacios ni caracteres especiales al inicio).

    
         console.log(miObjeto.nombre); // Salida: Producto A
         console.log(miObjeto.precio); // Salida: 25.99

    Notación de corchetes ([]):Es más flexible. Se utiliza cuando el nombre de la propiedad está almacenado en una variable o contiene espacios o caracteres especiales. También puedes usarla con cadenas dinámicas.

    
         console.log(miObjeto["disponible"]); // Salida: true
        const propiedad = "precio";
        console.log(miObjeto[propiedad]); // Salida: 25.99
  • this dentro de objetos La palabra clave this dentro de un método de un objeto se refiere al propio objeto que llamó al método. Es importante notar que el valor de this puede cambiar dependiendo de cómo se llama la función. Sin embargo, dentro de los métodos de un objeto definidos de la manera tradicional, this generalmente se refiere al objeto.
    • 
         const otroObjeto = {
                                marca: "Samsung",
                                modelo: "Galaxy S21",
                                descripcion: function() {
                                return `Este es un ${this.marca} ${this.modelo}.`;
                            }
                    };
        console.log(otroObjeto.descripcion()); // Salida: Este es un Samsung Galaxy S21.
