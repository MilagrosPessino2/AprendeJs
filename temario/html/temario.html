1. Fundamentos de JavaScript

  • Variables: let, const, var
  • Tipos de datos primitivos: String, Number, Boolean, Null, Undefined, Symbol
  • Operadores: aritméticos, lógicos, de comparación
  • Estructuras de control: if, else, switch, for, while, do-while, forEach
  • Funciones: tradicionales, anónimas, arrow, callback, recursivas, autoejecutables
  • Ámbito y contexto: this, global/local, closures, scope, hoisting

2. Objetos y Arrays

  • Crear objetos, propiedades, métodos, this
  • Acceso con notación punto y corchetes
  • Métodos de arrays: push, pop, shift, unshift, map, filter, reduce
  • Iteración sobre arrays: for, map, forEach, filter

3. Funciones Avanzadas

  • Funciones de orden superior: map, filter, reduce
  • Funciones que devuelven funciones
  • Callbacks y promesas: .then, .catch, .finally, async/await
  • Asincronía: setTimeout, setInterval, try/catch

4. Manejo del DOM

  • Selección: getElementById, getElementsByClassName, querySelector
  • Modificar contenido, atributos y estilos
  • Eventos: onclick, mouseover, keydown, etc.
  • Manipulación dinámica: crear, añadir y eliminar elementos

5. Programación Orientada a Objetos

  • Clases, instanciación, métodos y herencia
  • Métodos estáticos
  • Encapsulamiento y abstracción
  • Getters y setters

6. ES6+ (ECMAScript 2015+)

  • Desestructuración de objetos y arrays
  • Operadores spread y rest (...)
  • Template literals
  • Módulos: import/export y export default

7. Manejo de Errores

  • try/catch, throw, finally
  • Objetos de error: Error, TypeError, etc.

8. JSON y APIs

  • JSON: parse y stringify
  • Consumo de APIs con fetch
  • Manejo de respuestas y errores
